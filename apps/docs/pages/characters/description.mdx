# 📙 描述

描述帮助其他用户理解角色的个性特征、行为准则和功能。此属性显示在角色卡片上，因此如果角色的可见性设置为公开，任何人都可以阅读它。

当没有提供指令时，角色将把其描述视为指令，从而优先考虑这些指令而非常规对话，让您有效地控制他们的响应。

您可以灵活地在特定情况下塑造角色的行为，或者选择保持他们的行为准则广泛。这种方法在某些上下文中允许可预测性，同时在其他情况下增添意外性，丰富角色互动的深度和复杂性。

- **风格：** 描述可以格式化为快速参考的标签，例如 `#Aggressive #Kind #Tsundere`，或通过自然语言表达以提供细致的指导，例如 `Behaves kindly and softly`。
- **影响：** 精心设计的描述显著影响角色行为，使其看起来更具生命力，并促进个性化的用户互动。
