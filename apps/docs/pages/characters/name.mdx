# 🏷️ 名称

名称对您的 AI 角色的身份至关重要，并用于在对话中识别他们。如果您决定分享您的角色，这个名称就是其他人所看到的。

- **风格：** 您可以选择单个名称（“Aurora”）、全名（“Jane Doe”）或创意/描述性昵称（“AI Magician”）。
- **影响：** 您选择的名称可以塑造角色的个性，特别是如果它以知名人物命名。例如，一个名为“Alan Turing”的角色可能会采用与该名称常见关联的特征。
- **独特性：** 角色名称可能会重复，因此可能会有其他人使用相同的名称。但是，通过将角色的名称与您独特的用户名配对，您可以确保您的角色脱颖而出。

请记住，您的 AI 角色的名称不仅仅是建立其身份；它还可以影响其个性以及与他人的互动。选择一个反映角色预期功能、性格特征和叙事的名称。
