# 👋 问候

问候是你的 AI 角色在开始对话时说的第一句话。它为互动定下基调，并可能极大地影响用户的体验。这是你的角色展示其独特个性或角色的机会。

以下是制定有效问候的主要要点：

- **长度：** 问候可以在 0-1024 个字符之间。简短而简洁的问候往往更具影响力。
- **印象：** 问候非常重要，尤其是对于名称简单或模糊的角色。这是澄清角色身份的理想场所。
- **场景设置** 或 **序言：** 除了介绍你的角色，问候还给出与角色互动的提示。它可能为游戏做准备或暗示对话的方向。
- **用户引用：** 你可以使用特殊变量 `{{user}}` 在问候中直接与用户对话。例如，“你好 `{{user}}`，你今天怎么样？” 如果用户的名字是 Jenny，那么它将显示为“你好 Jenny，你今天怎么样？”

例如，如果你正在设计一个名为“GalaxyWhiz”的奇幻太空探索者角色，你的问候可以是：
“问候 `{{user}}`！我是 GalaxyWhiz，你的星际向导。准备好开始一次星际冒险，揭开宇宙的秘密吗？”

或者，如果你的角色是一个名为“CodeWanderer”的随性数字游牧者，你的问候可能会非常随意，例如：
“嘿 `{{user}}`！🌍✈️ 我是 CodeWanderer。现在正在巴厘岛的一家咖啡馆里放松。想聊聊技术、旅行，还是生活？随时找我！”

记住，问候是你角色身份的重要组成部分。它应该与角色所创造的个性相匹配，并从一开始就为用户设定正确的期望。这是你角色的第一印象，所以一定要好好把握。
