# 生成图像

[为您的角色卡生成高质量、详细的图像。](https://openroleplay.ai/images)

## 稳定扩散模型的工作原理

稳定扩散模型从文本提示中生成图像。它们擅长使用详细和清晰的语言来制作图片。

将您的提示视为指令，而模型则是绘图者。

## 如何编写一个优秀的提示

### 具体而详细

- **清晰的描述：** 用你的语言描绘出详细的画面。与其使用模糊的“一个人”，不如深入具体，比如他们的风格、年龄和独特特征。
  - **例子：** “一个活泼的男孩，留着尖锐的黑色短发，穿着霓虹色的卫衣、破洞牛仔裤和涂鸦满满的运动鞋，手里紧握着一块滑板。”
- **环境和氛围：** 勾勒出角色周围的场景。可以是“傍晚时分，坐落在涂鸦覆盖的屋顶上”或“徘徊在热闹的霓虹灯夜市中”。
- **情绪和语气：** 设置情感场景。诸如“宁静的低语”或“无拘无束的喜悦的迸发”这样的描述有助于塑造你图像的情感氛围。

## 可见性

图像默认是公开的，但可以设置为私密。
